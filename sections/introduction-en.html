Introduction back to ToC

This ontology has a relatively small amount of classes and properties, though it allows a complete description of digital objects. The aim of DPO concerns the development of a model to describe the main features of digital objects which can be stored or possessed by any institution. The description of them takes into consideration all the major factors and the possible relationships among files. This ontology is a first version, which means it can be considerably improved by anyone highlighting the need of increasing the already present descriptions.
